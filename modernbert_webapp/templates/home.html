About the Project - Document Classifier

The Document Classifier is an advanced document classification system powered by artificial intelligence. The project utilizes neural networks and Machine Learning models to analyze text and determine whether a document contains sensitive content or not.

Its goal is to offer an efficient and automated solution for document categorization, allowing users to upload files in PDF, TXT, or JSON format and receive a detailed analysis with metrics such as precision, recall, and accuracy.

Additionally, the project includes an administrative panel for managing trained models, enabling training of new models and customization of the system according to user needs.

This project was developed with a focus on efficiency and accessibility, aiming to make document analysis and categorization more streamlined and accurate.
